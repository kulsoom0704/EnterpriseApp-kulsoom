# Enterprise App - Kulsoom

## Project Description

This project is a prototype website for an NGO based in Anderlecht. The application was developed using Spring Boot and Thymeleaf. The website allows visitors to view events, see event details, create new events, view information about the NGO, and contact the administrators.

## Features

### Home Page (Index)

* Displays an overview of the latest events.
* Shows event title and organization.
* Provides a link to view event details.
* Contains a button to create a new event.

### New Event

* Form for creating a new event.
* Validation of required fields.
* Email validation.

### Event Details

* Displays all information related to a selected event.

### About

* Displays information about the NGO.
* Includes address, contact information, and description.

### Contact

* Contact form for visitors.
* Allows sending messages to administrators.
